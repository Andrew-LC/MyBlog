# Notion Powered Blog 

![Blog Post](./public/picutures/blogpost.png)

A simple Blog portfolio made using next js and notion to create a smoothless experience.

## Tech Stack
- Next JS 
- Tailwind CSS
- Notion API
- React-Markdown
- Prism JS 
- React-Synatx-Highlighter
- RehypeRaw 
- Notion to MD
